This plugin monetarizes your blog or website with adf.ly.

== Description ==

This plugin inserts the website entry script from adf.ly into your blog or website. With it you can earn money with each visitor of your page. Please be kind and use my referral link to register at adf.ly: http://adf.ly/?id=3757920

== Installation ==

* register at adf.ly under http://adf.ly/?id=3757920
* activate the plugin
* go to Settings -> Adfly
* insert your adf.ly ID
* have visitors on your site
* get paid out by adf.ly
